## 프로젝트 실행 방법

1. 프로젝트 클론

```bash
   git clone https://github.com/repo-story/repostory.dev
   cd repostory.dev/client
```

2. 종속성 설치

```bash
npm install
```

3. 개발 서버 실행

```bash
npm run dev
```
